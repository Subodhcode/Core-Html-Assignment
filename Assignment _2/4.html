List of HTML Tag

  1. html: The root element of an HTML document that encapsulates the entire content.
  2. head: Contains meta-information about the document, such as the title, character encoding, and links to external resources like stylesheets and scripts.
  3. meta: Provides metadata about the document, including character encoding and authorship details.
  4. title: Sets the title of the web page, displayed in the browser's title bar or tab.
  5. link: Links to external resources, such as stylesheets or icons, to enhance the presentation and functionality of the page.
  6. script: Embeds or links to JavaScript code that adds interactivity and functionality to the web page.
  7. style: Contains CSS (Cascading Style Sheets) rules for defining the page's visual presentation.
  8. body: Encloses the main content of the web page, including text, images, and other media.
  9. h1 - h6: Headings that define the hierarchical structure of the page, with h1 being the highest level and h6 the lowest.
  10. p: Represents a paragraph of text.
  11. a: Creates hyperlinks to other web pages, resources, or email addresses.
  12. img: Embeds images in the document, enhancing visual content.
  13. ul: Defines an unordered list, typically displayed with bullet points.
  14. ol: Defines an ordered list, with numbered items.
  15. li: Represents list items within ul or ol elements.
  16. div: A versatile container for grouping and styling content, often used in conjunction with CSS.
